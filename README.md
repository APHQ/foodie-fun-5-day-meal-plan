# Foodie Fun: 5-Day Budget Meal Plan

This repository contains the **5-Day Budget Meal Plan** project for the **BUS 333 Digital Marketing class**. The goal of this project is to create a **responsive, visually appealing website** that provides budget-friendly meal ideas and a **simple signup form** for users to receive the meal plan via email.

## Features
- **5-Day Meal Plan Page:** HTML and CSS layout with images for each recipe.
- **Responsive Design:** Works on mobile, tablet, and desktop devices.
- **Recipe Details:** Ingredients, cost per serving, and step-by-step method.
- **Email Signup Integration:** Uses EmailJS to send the 5-Day Budget Meal Plan to users after signup.
- **Styling:** Light red, orange, and white theme matching the Foodie Fun brand.
